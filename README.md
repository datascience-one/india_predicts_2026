# India Predicts 2026

### India’s First National Election Prediction Challenge

**Powered by Data Science Academy**

---

## 🚀 Overview

**India Predicts 2026** is a national-level data science challenge designed to test whether **elections can be predicted using structured data, analytics, and AI**—instead of opinion or intuition.

This is not a polling exercise.
This is a **public, verifiable test of analytical skill**.

Participants are required to analyze historical data, public datasets, and trends to predict election outcomes across **multiple Indian states**.

---

## 🧠 The Challenge

* Predict outcomes across **824 constituencies**
* Cover elections across **5 states**
* Use **only publicly available data sources**
* Submit predictions in a structured format
* Compete for accuracy, not opinions

---

## 🗺️ States Covered

* Assam
* Kerala
* Puducherry
* Tamil Nadu
* West Bengal

## 📊 Submission Format

Participants must submit predictions using the official Excel template.

### 📁 Workbook Structure

The submission file contains separate sheets for each state:

* Assam
* Kerala
* Puducherry
* Tamil Nadu
* West Bengal

---

### ✍️ How to Enter Predictions

* Only edit **Column F – Predicted Outcome**
* Allowed values:

  * `W` = Win
  * `L` = Lose
  * `O` = Other

⚠️ Enter only **one character per row**

---

### ❌ What NOT to Modify

* Candidate names
* Constituency names
* Column headers
* Sheet names
* Workbook structure

Any modifications may result in rejection.

---

### 📛 File Naming Format

```
IP2026_[email_ID]_[FullName].xlsx
```

**Example:**

```
IP2026_johndoe@gmail.com_JohnDoe.xlsx
```
